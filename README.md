# Анализ датасета Amazon (EDA & NLP)

Небольшой проект по предобработке данных, разведочному анализу (EDA) и частотному анализу текстов отзывов на основе датасета товаров с Amazon.

## Стек технологий
- **Python**
- **Обработка данных:** Pandas, NumPy, RegEx (модуль `re`)
- **Анализ текста (NLP):** NLTK, collections.Counter
- **Визуализация:** Matplotlib, Seaborn

## Что было сделано

### 1. Очистка и подготовка данных (Data Cleaning)
Сырые данные содержали лишние символы в числовых колонках (знаки валют, проценты).
- Настроена автоматическая очистка числовых колонок (`discounted_price`, `actual_price`, `discount_percentage`, `rating`, `rating_count`) с помощью регулярных выражений (оставлены только цифры и точки).
- Выполнено приведение типов к числовым форматам.
- Заполнены пропуски (NaN): числовые колонки заполнены нулями, строковые — значением 'Unknown'.
- Из длинной строки с категориями извлечена главная категория товара (`main_category`) для группировки.

### 2. Разведочный анализ (EDA)
- Вычислен средний размер скидки по основным категориям товаров (лидеры: Home Improvement, Computers & Accessories).
- Подсчитана популярность категорий (по сумме отзывов).
- Проверена гипотеза о связи размера скидки и итогового рейтинга товара. Вычислена корреляция (-0.13), которая показала отсутствие прямой зависимости.
- Построена диаграмма рассеяния (Scatterplot) распределения скидок и рейтингов.

### 3. Текстовая аналитика негативных отзывов (NLP)
Проведен анализ содержания плохих отзывов, чтобы выявить главные проблемы покупателей:
- Отфильтрованы товары с рейтингом ниже 3.5.
- Базовый словарь стоп-слов библиотеки NLTK расширен на 14 кастомных слов-паразитов (например: *'good', 'product', 'amazon', 'buy'* и т.д.), чтобы исключить их из анализа.
- Текст отзывов переведен в нижний регистр и разбит на отдельные слова с помощью регулярных выражений.
- С помощью `Counter` подсчитаны Топ-15 самых частых слов в негативных отзывах (чаще всего жалуются на *quality, remote, working*).
- Результат визуализирован с помощью горизонтальной столбчатой диаграммы (Barh) с пользовательским оформлением (кастомные цвета для топ-3, удаление рамок, подписи значений).

## Как запустить проект
1. Склонируйте репозиторий.
2. Убедитесь, что у вас установлены необходимые библиотеки (`pip install pandas numpy matplotlib seaborn nltk`).
3. Положите файл `amazon.csv` в корневую папку.
4. Откройте файл `pet_analyze.ipynb` в Jupyter Notebook или JupyterLab и выполните все ячейки.
