AMAZON PRODUCT REVIEW ANALYSIS

Анализ отзывов покупателей Amazon

О ПРОЕКТЕ
---------
Пет-проект по анализу датасета товаров и отзывов Amazon. Цель — выявить 
ключевые проблемы покупателей и закономерности между характеристиками товаров.

ЧТО БЫЛО СДЕЛАНО
----------------
1. Предобработка данных:
   - Очистка числовых колонок (цены, скидки, рейтинги) от специальных символов
   - Обработка пропущенных значений
   - Извлечение основных категорий товаров

2. Исследовательский анализ:
   - Анализ корреляции между скидкой и рейтингом
   - Визуализация распределения данных
   - Статистический анализ метрик

3. Анализ текстовых отзывов:
   - Обработка отзывов с низким рейтингом (< 3.5)
   - Удаление стоп-слов (NLTK)
   - Частотный анализ слов в негативных отзывах
   - Выявление ключевых проблем покупателей

ТЕХНОЛОГИИ
----------
- Python 3.7+
- Pandas — работа с данными
- NumPy — числовые операции
- Matplotlib & Seaborn — визуализация
- NLTK — обработка естественного языка
- Regular Expressions (re) — очистка текста
- Collections.Counter — частотный анализ

УСТАНОВКА
---------
1. Установите Python (если нет): https://www.python.org/downloads/

2. Создайте папку для проекта и поместите туда файлы:
   - pet.ipynb
   - amazon.csv
   - README.txt

3. Установите зависимости:
   pip install pandas numpy matplotlib seaborn nltk

4. Загрузите стоп-слова NLTK (один раз):
   import nltk
   nltk.download('stopwords')

ЗАПУСК
------
1. Откройте Jupyter Notebook:
   jupyter notebook pet.ipynb

2. Запускайте ячейки последовательно

3. Проект автоматически:
   - Загрузит и обработает данные
   - Построит визуализации
   - Проанализирует тексты отзывов
   - Сгенерирует итоговый график

ОСНОВНЫЕ РЕЗУЛЬТАТЫ
-------------------
Топ проблем покупателей (по отзывам с низким рейтингом):

1. Quality (53 упоминания) — качество товаров
2. Remote (41) — проблемы с пультами
3. Working (40) — проблемы с работой устройств
4. Price (24) — цена
5. Bluetooth (24) — проблемы с подключением
6. Work (18) — работа устройств
7. TV (17) — телевизоры
8. Best (17) — сравнения
9. USB (17) — проблемы с USB
10. Works (17) — функциональность
11. Charging (16) — зарядка
12. Using (16) — использование
13. First (16) — первое впечатление
14. Without (15) — отсутствие функций
15. Battery (15) — батарея

КЛЮЧЕВЫЕ ИНСАЙТЫ
----------------
- Основные жалобы связаны с качеством продукции
- Много негатива об электронике и аксессуарах
- Проблемы с беспроводным подключением (Bluetooth)
- Вопросы соотношения цены и качества
